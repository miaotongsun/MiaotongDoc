# 妙通文档 — 产品功能规划

---

## 一、产品定位

**妙通文档**：面向银行的企业文档协同平台

**核心价值**：让企业文档管得住、找得到、用得好

**用户画像**：

| 角色 | 需求 | 使用频率 |
|------|------|---------|
| 普通员工 | 创建/编辑/查找文档，发起审批 | 每天 |
| 部门经理 | 审批文档，管理团队文档 | 每天 |
| 合规/风控 | 查看审计日志，监控文档安全 | 每周 |
| IT管理员 | 管理用户/部门/系统配置 | 每周 |
| 高层领导 | 查看统计报表，签批重要文件 | 每周 |

---

## 二、系统模块总览

```
妙通文档
│
├── 1. 文档管理      ← 核心模块，所有功能围绕文档展开
│   ├── 文件夹
│   ├── 分类体系（业务类型 + 密级）
│   ├── 标签
│   ├── 模板库
│   ├── 搜索
│   ├── 回收站
│   └── 上传/下载
│
├── 2. 在线编辑      ← 核心能力，基于 OnlyOffice
│   ├── 文档编辑（Word）
│   ├── 表格编辑（Excel）
│   ├── 演示编辑（PPT）
│   ├── 评论批注
│   ├── 版本管理
│   └── 导出/打印
│
├── 3. 协同工作      ← 让文档流转起来
│   ├── 文档分享
│   ├── 审批流程
│   ├── 签署流程
│   ├── 任务指派
│   └── 在线感知
│
├── 4. 合同管理      ← 业务专项模块
│   ├── 合同台账
│   ├── 合同全生命周期
│   ├── 到期提醒
│   └── 合同统计
│
├── 5. 通知中心      ← 连接所有模块的消息枢纽
│   ├── 站内通知
│   ├── 待办提醒
│   └── 通知设置
│
├── 6. 管理后台      ← 系统管理
│   ├── 仪表盘
│   ├── 用户管理
│   ├── 部门管理
│   ├── 系统配置
│   └── 安全审计
│
└── 7. 安全体系      ← 贯穿所有模块
    ├── 权限控制
    ├── 文档水印
    ├── 操作审计
    └── 登录安全
```

## 三、各模块功能清单

> 所有文档的家，分类清晰、查找方便

## 五、用户核心流程

### 流程一：日常文档工作

```
登录 → 首页看到最近文档和待办
  → 打开文档，在线编辑
  → 添加评论，@同事讨论
  → 保存，自动创建版本
  → 分享给领导审批
  → 审批通过，文档状态更新
```

### 流程二：合同管理

```
新建合同 → 填写信息 → 关联文档/上传附件
  → 提交审批 → 审批人审核 → 通过
  → 各方签署 → 合同生效
  → 履行跟踪 → 到期提醒
  → 续签或归档
```

### 流程三：管理运维

```
管理员登录 → 管理后台
  → 查看仪表盘（用户/文档/存储统计）
  → 管理用户（创建/禁用/重置密码）
  → 管理部门（调整组织架构）
  → 查看审计日志（操作追溯）
  → 调整系统配置（安全策略/存储配额）
```

## 六、优先级排序

### P0 — 必须有（系统能用的基础）

```
文档管理：文件夹、分类、搜索、上传/下载
在线编辑：Word/Excel/PPT、自动保存、评论、版本
协同工作：文档分享、审批流程、签署流程
合同管理：合同台账、新建、编号、状态流转
通知中心：通知列表、未读标记
管理后台：用户管理、部门管理、操作日志
安全体系：权限控制、操作审计
```

### P1 — 应该有（提升体验）

```
文档管理：标签、模板库、回收站、收藏、最近访问
在线编辑：版本对比、导出PDF、打印、水印
协同工作：分享链接、签署取消、任务指派
合同管理：附件管理、变更记录、到期提醒、统计
通知中心：待办聚合、通知类型
管理后台：仪表盘、系统配置、日志导出
安全体系：密级管理、分享安全
```

### P2 — 锦上添花（后续迭代）

```
文档管理：批量操作、排序视图
在线编辑：版本对比详细
协同工作：超时升级、审批委托、审批模板、批量签署
合同管理：归档、导入导出
管理后台：趋势图、存储设置
安全体系：登录安全（二次验证）
```

### H. 设计风格参考（飞书 UI 规范）

```
飞书设计语言核心特征：

1. 颜色
   ├── 主色：#3370FF（飞书蓝）
   ├── 成功：#34C759
   ├── 警告：#FF9500
   ├── 错误：#FF3B30
   ├── 背景：#F5F6F7（页面）、#FFFFFF（卡片）
   └── 文字：#1F2329（标题）、#646A73（辅助）

2. 间距
   ├── 基础单位：4px
   ├── 组件内间距：8px-12px
   ├── 卡片间距：16px
   └── 区域间距：24px-32px

3. 圆角
   ├── 小组件：4px-6px
   ├── 卡片：8px-12px
   ├── 弹窗：12px
   └── 按钮：6px-8px

4. 阴影
   ├── 卡片：0 1px 2px rgba(0,0,0,0.04)
   ├── 悬浮：0 4px 12px rgba(0,0,0,0.08)
   └── 弹窗：0 8px 24px rgba(0,0,0,0.12)

5. 字体
   ├── 标题：16px-20px，font-weight: 600
   ├── 正文：14px，font-weight: 400
   ├── 辅助：12px，颜色 #8F959E
   └── 字体栈：-apple-system, PingFang SC, Microsoft YaHei

6. 动效
   ├── 过渡时间：0.2s-0.3s
   ├── 缓动函数：ease-out
   ├── 卡片悬浮：translateY(-2px) + shadow 加深
   └── 侧边栏展开：slide + fade

7. 图标
   ├── 风格：线性图标（2px 描边）
   ├── 大小：16px（行内）、20px（导航）、24px（标题）
   └── 颜色：跟随文字颜色或主题色
```
